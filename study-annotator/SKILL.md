# 学习批注HTML生成器

将PPT讲义、课程文档转化为集成康奈尔笔记、间隔复习标记和思维导图的双栏学习批注HTML文件。

## 工作流程

### Step 1: 信息收集

若用户未提供以下信息，先询问：
- **课程名称**（如：统计学、宏观经济学、有机化学）
- **章节主题**（如：第三章 假设检验）
- **学习目的**：`备考` / `深度理解` / 两者兼顾（影响批注侧重：备考侧重考点标注，深度理解侧重原理解析）

若用户已上传文件，直接解析内容，无需等待额外确认。

### Step 2: 内容解析

从上传的PPT/PDF/文本中提取：
1. 各幻灯片/段落的核心内容（保留原文层级结构）
2. 识别五类内容维度（见高亮规范）
3. 划分章节边界（用于插入章节总结块）

### Step 3: 生成HTML

严格按照[HTML结构规范](#html结构规范)生成单个完整`.html`文件，输出到 `/mnt/user-data/outputs/[课程名]-[章节]-学习批注.html`。

## HTML结构规范

生成的HTML包含五个部分，详细规范见 `references/html-template.md`。

### 五类高亮维度
| 颜色 | 含义 | CSS类 |
|------|------|-------|
| 🟡 黄色 | 核心概念与定义 | `.hl-yellow` |
| 🔴 红色 | 重点要求 / 高频考点 | `.hl-red` |
| 🔵 蓝色 | 数据、参数、具体细节 | `.hl-blue` |
| 🟢 绿色 | 操作步骤 / 格式规范 | `.hl-green` |
| 🟣 紫色 | 方法论 / 底层原理 | `.hl-purple` |

### 右栏批注四要素（每段必含）
1. **段落功能 / Paragraph Role** — 内容类型 + 在知识结构中的位置
2. **康奈尔要点 / Key Notes** — 2-4个关键词/问题；逻辑重心；关联标注（Link to: xxx）
3. **记忆难点 / Memory Flags** — 易错点；记忆技巧；难度标注（🔴Hard / 🟡Medium / 🟢Easy）
4. **自测题 / Self-Test** — 1个核心问题 + 折叠答案

### 自测题格式
```html
Q: [问题]
<details><summary>查看答案</summary>[答案]</details>
```

### 章节总结块（每章末尾）
- 思维导图骨架（文字缩进，3层层级）
- 康奈尔三句话总结（What / So What / Now What）
- 间隔复习提示（2-3个重点检验概念）

### 底部全文总览区（两栏）
- 左列：重点速查清单（可勾选，红色高亮内容）
- 右列：概念索引（黄色高亮内容，中英对照，1句话定义）

## 设计规范

读取 `references/html-template.md` 获取完整CSS和HTML模板代码。

### 核心设计原则
- **导航栏**：深海军蓝（`#1B2A4A`）背景 + 白色字体
- **页面底色**：米白（`#FAF8F3`）
- **批注卡片**：左侧色边 + 对应维度浅背景色 + 圆角
- **章节总结块**：深海军蓝左边框 + 浅灰背景，视觉区分于普通批注
- **字体**：正文 Lora（衬线），界面/批注 IBM Plex Sans
- **响应式**：移动端自动切换单栏布局
- **Sticky章节导航条**：点击跳转各节

## 质量检查清单

生成前确认：
- [ ] 每个左栏段落都有对应的右栏批注（严格对齐）
- [ ] 每段批注包含四要素（段落功能/康奈尔要点/记忆难点/自测题）
- [ ] 所有自测题答案使用`<details>`折叠
- [ ] 每章结尾有总结块（思维导图+三句话+复习提示）
- [ ] 底部有全文总览区（速查清单+概念索引）
- [ ] 幻灯片编号标注（Slide N）
- [ ] 高亮维度颜色正确对应内容类型
- [ ] HTML文件完整可独立运行（无外部依赖失效）
